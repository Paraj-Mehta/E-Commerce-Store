# E-Commerce

A full-stack e-commerce application built using:

- React
- Bootstrap
- Node.js
- Express.js
- MongoDB
- Cloudinary
- JWT Authentication

## Features

- User Authentication
- Admin Dashboard
- Category Management
- Product Management
- Cart Functionality
- Cloudinary Image Uploads

## Installation

Backend:

npm install
npm start

Frontend:

npm install
npm run dev
